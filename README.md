## 功能


1. 获取教务网课程表, 自动更新周数
2. 获取教务网成绩信息
3. 搜索图书馆馆藏信息
4. 获取个人图书借阅信息
5. 导出课表到日历
6. 增加/修改 课程信息
7. 从 Excel 文件导入课表(正在开发)
